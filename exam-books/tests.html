Indien deze boodschap zichtbaar is, betekent dat dat er zich een fout heeft voorgedaan tijdens het inlezen van je student.js bestand.

  • Open de browserconsole.
    WindowsMacOS
    Chrome Ctrl + Shift + J Cmd + Opt + J
    Firefox Ctrl + Shift + K Cmd + Opt + K
    Safari Cmd + Opt + C
  • De foutmelding zou een hint moeten geven naar wat er verkeerd gelopen is. Let ook op het regelnummer waarop de fout zich voordoet.

Indien er geen foutmelding staat of de browser niet lijkt te reageren, betekent dit dat je vermoedelijk een oneindige lus hebt geïmplementeerd.

  • Verwijder je meest recent geschreven code uit student.js. Het gemakkelijkste is door het in commentaar te zetten.
  • Refresh de pagina. Mocht dit niet lukken, sluit je de tab en heropen je ze (Ctrl + Shift + T of Command + Shift + T).
  • De opgaves zouden opnieuw moeten verschenen zijn. Indien dit niet zo is, verwijder je meer code uit student.js tot alles terug werkt.
  • Herbekijk je code en tracht te ontdekken waarom je lus niet eindigt.
